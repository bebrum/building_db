# Словарь данных

| Таблица | Поле | Описание |
|---|---|---|
| `Марки` | `марка` | Название марки автомобиля, первичный ключ. |
| `Марки` | `категория` | Категория автомобиля: например, грузовая или легковая. |
| `Авто` | `ID_авто` | Уникальный идентификатор автомобиля. |
| `Авто` | `марка` | Марка автомобиля, связана с таблицей `Марки`. |
| `Авто` | `год_выпуска` | Год выпуска автомобиля. |
| `Авто` | `стоимость` | Стоимость автомобиля. |
| `Продавец` | `ID_продавца` | Уникальный идентификатор продавца. |
| `Продавец` | `ФИО` | ФИО продавца. |
| `Продавец` | `должность` | Должность сотрудника. |
| `Покупатель` | `ID_покупателя` | Уникальный идентификатор покупателя. |
| `Покупатель` | `ФИО` | ФИО покупателя. |
| `Покупатель` | `наличие_прав` | Есть ли водительские права: `есть` или `нету`. |
| `Покупатель` | `бюджет` | Бюджет покупателя. |
| `Покупатель` | `дата_рождения` | Дата рождения покупателя. |
| `Продажи` | `ID_продажи` | Уникальный идентификатор продажи. |
| `Продажи` | `ID_авто` | Автомобиль, который был продан. |
| `Продажи` | `ID_продавца` | Продавец, оформивший продажу. |
| `Продажи` | `ID_покупателя` | Покупатель, участвующий в продаже. |
| `Продажи` | `стоимость` | Стоимость продажи. |
